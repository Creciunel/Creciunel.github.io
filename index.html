Temperature Control

A temperature control device by controlling 220V voltage on ESP32 with a web interface, triac, and ZDC.

Software components:

  • Auto-tune PID controller for precise temperature control
  • FreeRTOS for efficient task management and scheduling
  • WiFi server with memory storage for providing real-time data

Hardware components:

  • BC547 transistor as intermediate driver
  • MOC3052 optotriac for safe 220V switching
  • BTA24 (or BT138) triac for resistive loads
  • Zero-cross detection using PC817 optotransistor
